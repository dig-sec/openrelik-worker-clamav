Welcome to Utgard

Utgard is a secure, isolated malware analysis lab providing forensic analysis capabilities, evidence processing workflows, and controlled execution environments. The platform integrates 15 specialized workers for comprehensive artifact analysis.

Key Capabilities:

  • Forensic artifact analysis and timeline construction
  • Malware detection, classification, and capability analysis
  • Windows registry and file system forensics
  • Email and document forensics
  • Network monitoring and packet capture analysis
  • Isolated execution environment with controlled egress

üöÄ Getting Started

First time setup:

  1. Run provisioning: ./scripts/provision.sh
  2. Test connectivity: ./scripts/test-connections.sh
  3. Access OpenRelik at http://localhost:8711/
  4. Upload evidence and create analysis tasks

Useful commands:

  • Check status: ./scripts/check-status.sh
  • Start lab: ./scripts/start-lab.sh
  • View logs: docker-compose logs -f
